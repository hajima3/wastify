# Wastify Frontend

## Setup

1. Install dependencies:
   ```
   npm install
   ```
2. Run the development server:
   ```
   npm run dev
   ```

## Pages
- Landing page
- Login / Register
- Household Dashboard
- Municipal Dashboard
- Rewards
- Waste Guide
- Admin Analytics
- Route Map

## Theming
- TailwindCSS green palette
